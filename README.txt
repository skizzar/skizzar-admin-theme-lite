== Description ==

Skizzar Admin Theme Lite is a beautifully reimagined admin theme for Wordpress and Wordpress Multisite. It's modern approach to design will transform your dashboard into a more inviting, better looking and more intuitive interface.

Features include:
- Horizontal menu (with "More" tab for items that don't fit on the menu)
- Attractive Banner under menu with profile details
- Modern User interface
- Improved responsive design

Upgrade to Pro to unlock more features, including:
- Styled login page
- Styled site toolbar
- Custom banner image
- Show/hide almost all dashboard features based on user role
- Notification Center
...and more to come!

Give your wordpress dashboard a much needed makeover, transform your admin interface with Skizzar Admin Theme Lite

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Your plugin is good to go, traight out of the box
4. Edit plugin settings in Appearance > Admin Theme

== Frequently Asked Questions ==

= Are there any more features available? =

Yes, we have MUCH more feature in our Pro version. Head over to http://wordpressadmintheme.skizzar.com to download it.

= Can I change the banner image? =

Unfortunately not in our Lite version - check out our Pro version though.

== Is Skizzar Admin Theme compatible with my other plugins?

We've done our best to try and make Skizzar Admin Theme compatible with all common wordpress plugins - we may well have missed something though, so if you find something that doesn't look quite right, get in touch.
